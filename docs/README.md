## Navigation Tips

1. **New to the project?** Start with `/docs/START-HERE.md` or `/docs/GETTING-STARTED-NOW.md`
2. **Need to deploy?** Check `/docs/deployment/`
3. **Working on a specific module?** Navigate to the component's `/docs` folder
4. **Looking for weekly progress?** See `/docs/weekly-reports/`
5. **Debugging build issues?** Check `/docs/development/`

## Maintenance
